Image Slideshow App
Description:

A simple desktop application that displays images in a slideshow format

Built using Python's Tkinter library for GUI

Features:

Displays multiple images in sequence

Auto-resizes all images to 1080x1080 pixels

3-second pause between each image

Click "Play SlideShow" button to start

Automatically loops through all images

Libraries Used:

tkinter - For creating the GUI window

PIL (Pillow) - For image handling and resizing

itertools - For cycling through images

time - For controlling slideshow speed
